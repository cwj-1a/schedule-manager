# 排班登记管理系统

一个简单易用的排班管理工具，支持自定义登记任意年份、月份、日期、星期几，安排某人在特定时间段工作。

## 功能特点

- 日期选择：支持选择任意年份（1900-2100）、月份和日期
- 星期自动计算：根据选择的日期自动计算星期几
- 人员管理：可输入任意人员姓名
- 时间段设置：支持设置开始时间和结束时间
- 备注功能：可为每条排班添加备注信息
- 数据持久化：使用localStorage保存数据，刷新页面不丢失
- 筛选功能：支持按年份、月份、人员筛选排班记录
- 冲突检测：自动检测同一人员的时间段冲突
- 删除功能：支持删除不需要的排班记录
- 响应式设计：支持桌面端和移动端
- 标签页切换：新增排班和历史排班两个视图
- 历史排班查看：以表格形式展示所有排班记录
- 统计信息：自动计算总排班数、参与人数、总工时
- 导出功能：支持导出为CSV、Excel、HTML三种格式

## 使用方法

### 1. 打开应用

直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 添加排班

1. 在左侧表单中填写信息：
   - 年份：输入或选择年份
   - 月份：从下拉菜单选择月份
   - 日期：输入具体日期（会自动更新星期）
   - 星期：自动根据日期计算，也可手动选择
   - 人员：输入人员姓名
   - 开始时间：选择工作开始时间
   - 结束时间：选择工作结束时间
   - 备注：可选，填写相关备注信息

2. 点击"添加排班"按钮提交

### 3. 查看排班

右侧列表会显示所有排班记录，按日期和时间排序。

### 4. 筛选排班

使用筛选功能可以快速查找特定排班：
- 年份筛选：输入特定年份
- 月份筛选：选择特定月份
- 人员筛选：输入人员姓名进行搜索
- 点击"筛选"按钮应用筛选条件
- 点击"重置"按钮清除所有筛选条件

### 5. 删除排班

点击排班记录右侧的"删除"按钮，确认后即可删除该条记录。

### 6. 历史排班查看

点击顶部"历史排班"标签页，可以查看所有排班记录的详细信息：

**统计信息卡片：**
- 总排班数：显示当前筛选条件下的排班总数
- 参与人数：显示参与排班的不同人员数量
- 总工时：自动计算所有排班的总工作时长

**历史排班表格：**
- 以表格形式展示所有排班记录
- 包含日期、星期、人员、工作时间、备注等信息
- 支持直接在历史视图中删除排班记录

**筛选功能：**
- 按年份、月份、人员筛选历史排班
- 筛选后统计数据和表格会实时更新

### 7. 导出排班表

系统支持三种导出格式，方便通知相关人员：

**CSV格式：**
- 通用格式，可用Excel、WPS等软件打开
- 适合数据分析和进一步处理

**Excel格式：**
- 直接生成.xls格式文件
- Excel原生支持，无需转换

**HTML格式：**
- 生成美观的网页格式
- 包含完整的样式和统计信息
- 可直接打印或分享
- 自动统计每人排班次数和总工时

**导出说明：**
- 可以先使用筛选功能选择要导出的数据范围
- 导出的数据会按日期和时间自动排序
- HTML格式包含更丰富的统计信息和样式

## 技术特点

- 纯前端实现，无需后端服务器
- 使用localStorage进行数据存储
- 响应式布局，适配各种屏幕尺寸
- 现代化UI设计，操作简单直观
- 自动验证输入数据的有效性

## 注意事项

- 数据存储在浏览器的localStorage中，清除浏览器数据会丢失所有排班记录
- 建议定期备份重要数据
- 同一人员在同一时间段不能重复排班
- 结束时间必须晚于开始时间

## 浏览器兼容性
